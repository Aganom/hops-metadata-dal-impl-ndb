Hops Metadata DAL NDB Implementation
===

Hops Database abstraction layer for storing the hops metadata in MySQL Cluster

How to build
===

```
mvn clean install
```

deploys the jar file as an artifact to the kompics maven repository.

```
./deploy.sh
```
